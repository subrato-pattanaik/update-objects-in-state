## Tech stack used for this app

- Vite for running and building app.
- Spectacle for creating presentation slides
- Shadcn for ui components
- Tailwind for css

## Get it running locally

- Clone or fork it.
- Run `pnpm install`.
- Run `pnpm run dev`.
